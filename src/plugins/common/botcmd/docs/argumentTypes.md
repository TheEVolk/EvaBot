# Стандартные типы
>  Параметры помеченные звёздочкой являются обязательными.

## Integer
>  Целое число.

| Параметр | Описание                   |
|----------|----------------------------|
| max      | Максимальное число         |
| min      | Минимальное число          |

## String
>  Текст с пробелами.

| Параметр | Описание                   |
|----------|----------------------------|
| max      | Максимальная длина строки  |
| min      | Минимальная длина строки   |

## String
>  Текст с пробелами.

| Параметр | Описание                   |
|----------|----------------------------|
| max      | Максимальная длина строки  |
| min      | Минимальная длина строки   |

## Word
>  Строка без пробелов

| Параметр | Описание                   |
|----------|----------------------------|
| max      | Максимальная длина строки  |
| min      | Минимальная длина строки   |
